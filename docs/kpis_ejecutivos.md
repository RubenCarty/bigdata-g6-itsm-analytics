# KPIs Ejecutivos — Marco ITIL para Dashboard CIO/CTO

## KPIs de Velocidad
- **MTTR** (Mean Time To Resolve): objetivo < 4h prioridad alta
- **MTTA** (Mean Time To Acknowledge): objetivo < 30 min prioridad crítica

## KPIs de Calidad
- **SLA Compliance**: objetivo > 95% general
- **FCR** (First Contact Resolution): objetivo > 70%
- **Reopen Rate**: tickets reabiertos / total < 5%

## KPIs de Volumen
- **Ticket Volume Trend**: variación mensual vs mismo mes año anterior
- **Backlog Rate**: tickets abiertos > SLA / total abiertos < 10%

## KPIs de Negocio (nuevos — propuesta del proyecto)
- **Brecha Index**: incidencias recurrentes / total < 10%
- **TI Capacity Ratio**: horas incidencias / horas totales equipo TI
- **Proactive vs Reactive**: tickets detectados vs reportados por usuarios
