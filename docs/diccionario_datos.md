# Diccionario de Datos — Tickets ITSM

## Catálogo de Categorías ITIL

| Categoría | Subcategorías ejemplo |
|-----------|----------------------|
| Red/Conectividad | VPN, WiFi, Internet, LAN, Firewall |
| Hardware | PC, Laptop, Impresora, Servidor, Monitor |
| Software | Instalación, Error, Licencia, Actualización |
| Accesos y Seguridad | Contraseña, Permisos, AD, MFA |
| Correo/Colaboración | Outlook, Teams, Gmail, SharePoint |
| ERP/SAP | Módulo FI, MM, HR, SD, Error transacción |
| Base de Datos | SQL Server, Oracle, PostgreSQL, Backup |
| Infraestructura | Servidor, Storage, Virtualización, Cloud |
| Telefonía | IP Phone, Celular corporativo, Central |
| Proyectos/Requerimientos | Nuevo sistema, Mejora, Integración |

## Estados del Ticket
- `nuevo` → `asignado` → `en_proceso` → `pendiente_usuario` → `resuelto` → `cerrado`
- `reabierto` (desde resuelto/cerrado si el problema persiste)
